# PhishGuard Privacy Policy
**1. Data Collection**
PhishGuard does not collect, store, or transmit any Personally Identifiable Information (PII). 
**2. URL Analysis**
To provide real-time phishing protection, PhishGuard analyzes the URL of the active webpage tab. This analysis is performed by our secure cloud backend. URLs are hashed and are not tied to individual users or IP addresses.
**3. User Reports**
If a user voluntarily clicks "Report Site", the reported URL and the date are synced to a public, open-source repository to improve the global machine learning dataset. No user data is attached to these reports.
**4. Third-Party Services**
We do not sell, trade, or otherwise transfer your information to outside parties.
